En el repositorio de GitHub se está trabajando en el desarrollo de un juego de UNO simple, el cual actualmente se encuentra en su versión 1. En esta primera etapa se ha subido el código base junto con la documentación correspondiente y un diagrama UML que muestra cómo están conectadas las diferentes clases del sistema. Todo esto se ha realizado con el objetivo de mantener un control de calidad, registrar el progreso y asegurar la trazabilidad de los cambios.
Por el momento, el proyecto se encuentra a la espera de la versión 2, en la que se incorporarán las cartas especiales que añadirán nuevas dinámicas al juego. Esta futura entrega implicará ajustes en la lógica, la actualización de la documentación y del diagrama UML, así como la realización de pruebas adicionales para garantizar la estabilidad del sistema.
De esta manera, el repositorio no solo refleja el avance técnico del proyecto, sino también un proceso estructurado de desarrollo, asegurando que cada nueva versión se construya sobre una base sólida y bien documentada.
